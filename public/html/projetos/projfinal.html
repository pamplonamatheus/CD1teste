Projeto Final

Nome: Projeto Final

Curso: Engenharia Geral - Design de Software

Semestre: Primeiro

Data de início: 28/04/2016

Tempo de Duração do Projeto: Um mês

Design com foco no usuário: Sim

Descrição: Foi o último projeto de Design de Software. Neste projeto os alunos deveriam itegrar ao menos três dos seguintes itens: Firebase, Flask, HTML e TKinter. O resultado final variou de grupo para grupo. Foram criados jogos, aplicativos e sites neste projeto. Neste projeto, era preciso entregar algo que interagisse com seu usuário.

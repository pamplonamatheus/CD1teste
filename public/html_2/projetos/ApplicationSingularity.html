Application Singularity

Nome: Application Singularity

Curso: Engenharia Mecânica, Mecatrônica e Computação

Semestre: Primeiro

Data de início: 23/05/2016

Tempo de Duração do Projeto: Duas semanas

Design com foco no usuário: Não

Descrição: O projeto Application Singularity foi uma simulação do trabalho realizado pela Singularity University, o qual visa buscar soluções para os principais problemas encontrados em diferentes partes do mundo. A simulação foi realizada de maneira rigorosa sendo necessário escrever como seria a ideia para a solução de um problema da escolha do aluno além de gravar um vídeo explicando como seria implementado a solução para o problema escolhido. O projeto foi realizado na disciplina Grandes Desafios da Engenharia para todos os alunos de engenharia do primeiro semestre no Insper
